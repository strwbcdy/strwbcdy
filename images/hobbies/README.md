# Hobby Images

Place hobby-related images in this directory.

## Recommended Format
- **File format**: JPG or PNG
- **Dimensions**: 1920x1080 (16:9 aspect ratio) or similar
- **File size**: Optimize to < 500KB
- **Naming**: Use descriptive names like `photography.jpg`, `hiking.png`

## Usage
Reference these images in `src/data/portfolioData.ts` for hobby entries:
```typescript
image: "/images/hobbies/hobby-name.jpg"
```
