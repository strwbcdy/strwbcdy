# Company Logos

Place company logos and brand images in this directory.

## Recommended Format
- **File format**: PNG (with transparency) or SVG
- **Dimensions**: Square format (e.g., 400x400) or company logo dimensions
- **File size**: Optimize to < 200KB
- **Naming**: Use company names like `techcorp.png`, `startup-inc.svg`

## Usage
Reference these images in `src/data/portfolioData.ts` for experience entries:
```typescript
image: "/images/companies/company-name.png"
```
