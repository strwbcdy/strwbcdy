# Skill Icons and Images

Place skill-related icons and images in this directory.

## Recommended Format
- **File format**: PNG or SVG
- **Dimensions**: Square format (e.g., 300x300)
- **File size**: Optimize to < 100KB
- **Naming**: Use skill names like `frontend-dev.png`, `react-logo.svg`

## Usage
Reference these images in `src/data/portfolioData.ts` for skill entries:
```typescript
image: "/images/skills/skill-name.png"
```

## Tips
- Consider using technology logos from official sources
- Maintain consistent styling across all skill icons
