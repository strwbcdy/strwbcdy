# Favorite Music Album Covers

Place album cover images in this directory.

## Recommended Format
- **File format**: JPG or PNG
- **Dimensions**: Square format (e.g., 600x600)
- **File size**: Optimize to < 300KB
- **Naming**: Use artist and album names like `artist-album.jpg`

## Usage
Reference these images in `src/data/portfolioData.ts` for music entries:
```typescript
image: "/images/music/album-name.jpg"
```

## Alternative
You can also use external URLs from music services (ensure proper licensing).
